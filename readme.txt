To run the Project
————————--------

1. Go to mns_server directory from command line and run the following commands :

a. npm install —legacy-peer-deps
b. nodemon

The server will be running on port 3000
 
2. Go to the “mns_client” directory from command line and run the following commands:

a. npm install —legacy-peer-deps
b. npm start

3. Run the following command on Mac to start the MongoDB services using :

brew services start mongodb-community@6.0

To stop run:

brew services stop mongodb-community@6.0

4. Go to https://localhost:5001 to launch the web application and follow the instructions given in the report file and demo link to test the application
